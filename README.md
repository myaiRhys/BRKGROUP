# BRK Group Website

Lead generation and digital presence for trades & service businesses across South Africa.

## Structure

```
├── index.html          # Main website
├── css/
│   └── styles.css      # All styles
├── js/
│   └── main.js         # Animations & interactions
├── assets/             # Images & SVGs (add logo here)
└── Images/             # Brand assets
```

## Tech Stack

- HTML5
